GrowMap helps organize garden and balcony plantings with interactive maps, weather-aware recommendations, compatibility warnings, and analytics.

Demo flow

  1. Create a map with real plot sizes.
  2. Add plants, buildings, and zones.
  3. Log watering and harvest results.
  4. Check weather advice and analytics.

Key features

Private accounts and maps per user.
Interactive SVG map with click-to-inspect.
Plant catalog with care requirements.
Weather recommendations, not raw data.
Compatibility warnings for close plantings.
Harvest efficiency vs average yield.

Architecture

Frontend (GitHub Pages)

Static landing and documentation served from /docs. The app itself runs on Render.

Backend (Render)

Flask + SQLite, weather API integration, analytics, and activity logs.
